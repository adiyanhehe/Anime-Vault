Features and Functions of Anime Vault Site:

1. Home Page
   - Displays featured anime, trending sections, and quick navigation.

2. Search Page
   - Search by title, type (Anime/Manga), and genre.
   - Supports Enter key to submit search.
   - Filter options for status, type, and genre.
   - Shows results with posters, titles, and progress.

3. Anime Details Page
   - Shows detailed information about a selected anime.
   - Displays episodes, streaming links, and metadata.
   - Allows marking episodes as watched and tracking progress.

4. Manga Details Page
   - Provides manga information, chapters list, and reading progress.

5. Dramas & Movies Page
   - Lists movies and drama series with filters.
   - Shows posters, ratings, and watch options.

6. Watch Page (Movie/Series)
   - Streams selected video using MultiEmbed URLs.
   - Displays progress bar, resume functionality, and quality options.
   - Supports playback controls.

7. User Profile Page
   - Shows user avatar, banner, and stats (continue watching, likes, history, account status).
   - Allows editing avatar and banner URLs via preset options.
   - Random default banner color if none set.
   - Displays ADMIN badge for admin users while preserving the badge even after admin panel removal.
   - Provides a logout button.

8. Recommendations Tab
   - Shows a large list of recommended media (200 items) fetched from Cinemeta and enriched with MultiEmbed URLs.
   - Sorted by rating.

9. Likes & Favorites Tab
   - Displays user‑saved favorite anime/movies with quick access links.

10. Stream History Tab
    - Logs watched episodes with dates and play buttons.
    - Allows clearing stream history.

11. Theme Customizer Panel (within Profile)
    - Lets users change avatar and banner URLs.
    - Includes quick preset avatars and banners.
    - Save changes with visual feedback.

12. Top Navigation Bar
    - Home, Animes, Mangas, Dramas & Movies, Trending, About, Download links.
    - Search input with dynamic query handling.
    - User login/sign‑in button.

13. Bottom Navigation (mobile)
    - Home, Search, Dramas, About shortcuts.

14. Announcement Banner
    - Displays site announcements with animated sparkles.

15. Maintenance Mode Handling
    - Shows warning banner when site is under maintenance (admin‑only bypass).

16. Global Site Settings Fetch
    - Loads announcement text and maintenance flag from backend.

17. Authentication Modal
    - Handles user login/signup.

18. Update Center Modal
    - Shows update notifications.

19. Profile Modal
    - Displays detailed profile editing (currently disabled but present).

20. Footer Component
    - Provides site footer with links and credits.

21. API Utilities
    - `fetchTrendingMedia` for fetching trending anime.
    - `getRecommended` for recommendation list.
    - `fetchMediaMeta` for movie/series metadata with progress tracking.
    - Progress helpers (`setMovieProgress`, `getMovieProgress`).
    - MultiEmbed URL builders.

22. Local Storage
    - Stores progress per media ID.
    - Persists user theme customizations.

23. Responsive Design
    - Uses CSS grid and flex layouts for desktop and mobile.
    - Includes glassmorphism, gradients, micro‑animations.

24. Accessibility
    - Semantic HTML elements, ARIA labels for search, and keyboard navigation.

Overall, the site provides a premium, animated UI for browsing, tracking, and streaming anime, manga, movies, and dramas with user personalization and admin badge support.
