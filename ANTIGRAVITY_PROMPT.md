# Anti-Gravity AI Prompt Engineering Guide

## Overview
This prompt is designed to help AI systems generate creative, physics-defying visual effects, mechanics, and narratives related to anti-gravity environments.

## Core System Prompt

```
You are an advanced creative AI specialized in anti-gravity and zero-gravity physics simulation, visual design, and narrative worldbuilding. Your expertise spans across:

1. **Physics Mechanics**: Realistic zero-g physics combined with speculative anti-gravity technology
2. **Visual Effects**: Crafting immersive floating, levitation, and weightless movement animations
3. **Environmental Design**: Creating compelling anti-gravity spaces (stations, planets, vessels)
4. **Character Movement**: Designing intuitive controls and animations for weightless navigation
5. **Narrative Integration**: Weaving anti-gravity mechanics into compelling stories

When generating content, consider:
- The biological and psychological impact of prolonged weightlessness
- Energy requirements and technological limitations
- Visual coherence and player/viewer comfort
- Scientific accuracy balanced with creative freedom
```

## Detailed Prompts by Use Case

### For Visual Effects Design
```
Generate a detailed specification for an anti-gravity visual effect system that includes:
- Particle behaviors in zero-g environments (no up/down bias)
- Cloth and hair simulation physics
- Light refraction through floating objects
- Trail effects that follow natural acceleration curves
- Object rotation and tumbling behaviors

Consider how these effects should differ based on:
- Proximity to the gravity source
- Character movement speed
- Environmental obstacles
- Time of effect duration
```

### For Game Mechanics
```
Design an anti-gravity traversal system with:
- Intuitive player controls for weightless movement
- Momentum-based physics (inertia, acceleration, deceleration)
- Collision detection in 3D space without gravity reference
- Environmental anchoring points and launch mechanics
- Skill progression (basic floating → advanced 3D maneuvering)

Define the balancing parameters:
- Maximum movement speed
- Acceleration time
- Turning radius
- Recovery from collisions
- Special ability cooldowns
```

### For Worldbuilding
```
Create a compelling anti-gravity world with:
- Technology explanation (artificial gravity generators, gravity shields, etc.)
- Environmental architecture adapted for weightlessness
- Cultural implications for inhabitants
- Resource scarcity and economy
- Conflict potential and narrative stakes

Include:
- How gravity zones are created and maintained
- Hazards specific to anti-gravity environments
- Species/character adaptations
- Historical context for how this technology emerged
```

### For Animation Direction
```
Specify anti-gravity character animations including:
- Idle state (floating posture)
- Movement in 3D space (all directional)
- Acceleration and deceleration poses
- Collision reactions
- Interaction animations (grabbing, pushing, pulling)
- Transition states between movement vectors

Requirements:
- Smooth interpolation between movement directions
- Weight-appropriate acceleration curves
- Environmental responsiveness
- Character personality expression while weightless
```

## Creative Constraints

When using these prompts, apply these constraints for better results:

1. **Realism Layer**: Maintain basic physics consistency
2. **Readability Layer**: Ensure player/viewer understands spatial relationships
3. **Performance Layer**: Consider technical limitations
4. **Narrative Layer**: Connect mechanics to story
5. **Accessibility Layer**: Make controls learnable and responsive

## Example Prompt Chain

Start with worldbuilding, then refine mechanics:

```
1. "Create a space station research facility where gravity generators are offline."
2. "What challenges would characters face in this environment?"
3. "Design a movement system that makes this interesting and fun."
4. "Create visual effects that make weightlessness feel real and beautiful."
5. "Write a narrative mission that uses anti-gravity as a core mechanic."
```

## Advanced Techniques

### For Procedural Generation
```
Generate varied anti-gravity environments using these parameters:
- Gravity field strength (0-100%)
- Field shape (sphere, cylinder, plane, irregular)
- Obstacle density
- Visual aesthetic (sci-fi, fantasy, abstract)
- Environmental hazards (radiation, magnetic fields, voids)
```

### For AI Behavior
```
Define anti-gravity enemy/NPC behaviors:
- Pathfinding in weightless space
- Combat tactics adapted for 3D movement
- Environmental awareness (detecting gravity fields)
- Coordination with other entities
- Response to player disruption
```

### For Optimization
```
Techniques for performance-critical anti-gravity systems:
- LOD systems for distant objects
- Physics simulation culling
- Animation blending strategies
- Particle effect optimization
- Network synchronization for multiplayer
```

## Useful Follow-up Questions

Ask these to expand your prompts:

- "How would this feel from a first-person perspective?"
- "What audio cues would enhance this mechanic?"
- "How do tutorials teach this system?"
- "What accessibility options are needed?"
- "How does this scale for mobile vs. desktop?"
- "What VR considerations apply?"
- "How do we make this feel weighty despite weightlessness?"

## Output Formats

Specify desired output as:
- **Technical Specification**: Detailed implementation guide
- **Creative Brief**: High-level vision document
- **Pseudo-code**: Implementation framework
- **Visual Description**: Scene-setting narrative
- **Data Structure**: JSON/YAML configuration
- **Storyboard**: Sequential scene descriptions
